# Brazilian E-Commerce Sales Analysis (SQL)

# Database built locally from Kaggle CSVs using SQLite

## Objective
Analyze sales and revenue, customer behavior, order and fulfillment performance, product and category performance, and seasonal trends using the Olist Brazilian e-commerce dataset.

## Dataset
Public Brazilian e-commerce dataset containing orders, customers, products,
payments, and delivery information.

## Tools & Skills
- SQLite
- SQL (joins, aggregations, CASE logic, date functions)
- VS Code

## Key Business Questions
- Are there seasonal spikes in orders or revenue?
- How does delivery performance vary by product category?
- Which categories have the highest logistics cost relative to price?
- How do customer ordering patterns differ by region?

## Outputs
- SQL queries organized by business theme
- Insights and findings summarized in plain language
